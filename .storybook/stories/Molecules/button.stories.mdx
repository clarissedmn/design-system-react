# Button

#

#

#### Buttons allow to trigger an operation.

#

---

In OCTO Design System, we provide four types of button :

- Primary button
- Outline Button
- Link Button

‚ôøÔ∏èIf the button is not **explicit** ( = the title alone does not make it possible to understand its destination or its function), the `aria-label` attribute or the `title` attribute should then be used to specify the link destination or button function.

   **_Exemple :_**

  ```

  <Button kind="link" href="..." label="Link" aria-label="..." />
  ```
  ```

   <Button kind="link" href="..." label="Link" title="..." />

  ```
